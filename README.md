# nexclaw

A dual-mode CLI tool that operates both in terminal and via Telegram. Built with Bun and TypeScript for fast, efficient command-line interactions.

## Features

- **Dual Interface**: Seamlessly switch between CLI and Telegram modes
- **Stylized TUI**: Beautiful ASCII art banners with shadow effects
- **Interactive Prompts**: Modern command-line interface using Clack
- **Fast Runtime**: Powered by Bun for lightning-fast execution

## Installation

```bash
bun install
```

## Usage

Run the CLI:

```bash
bun run index.ts
```

Or use the wakeup command:

```bash
bun run index.ts wakeup
```

## Development

This project was created using `bun init` in bun v1.3.14. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
